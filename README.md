## 🔬 About Me

I am a **Bioinformatics MSc student at the University of Birmingham**, passionate about **cancer systems biology, multi-omics integration, and computational genomics**.  
With strong foundations in **molecular biology, genomics, and machine learning**, I build reproducible pipelines that bridge wet-lab biology with large-scale computational analysis.

I work on problems such as:

- Multi-omics-driven biomarker discovery  
- Deep learning for protein–ligand binding affinity  
- GWAS pipelines for polyploid crops  
- Structural variant (SV) analysis & trio-based inheritance workflows  
- lncRNA characterisation & regulatory network biology  
- HPC-based genomics workflows on BlueBEAR

# 🧪 Featured Research Projects

| **Project** | **Description** | **Tech & Methods** |
|-------------|-----------------|---------------------|
| **GWAS Pipeline for Autotetraploid Potato** | Pipeline integrating SNP data, phenotyping, DAPC/STRUCTURE-based population structure & BLUEs environmental correction to detect loci underlying canopy architecture & yield. | R · Linux · GWAS · DAPC · STRUCTURE |
| **lncRNA Gm26809 Characterisation** | In-silico & qPCR-based characterisation of melanoma-derived murine lncRNA Gm26809 and tissue-specific expression analysis. | RNAfold · CPC2 · LncRRISearch · qPCR |
| **MAPK Tri-Nodal Drug Repurposing** | Identifying therapeutics targeting PDPK1–ERK–mTOR triad using molecular docking & pathway modelling. | PyRx · AutoDock Vina · PBNs |
| **RLBindDeep** | ResNet–LSTM model for large-scale protein–ligand binding affinity prediction. | DL · TensorFlow · CNN-GRU |
| **PAM50GenePath** | CatBoost-driven PAM50 subtype classifier using pathway-aware gene expression signatures. | CatBoost · Gene Expression ML |
| **BioAlignNet** | GPU-accelerated global sequence alignment framework. | CUDA · Python |
| **SkinEnsemNet** | Multi-model ensemble deep learning pipeline for skin cancer classification. | CNNs · Ensemble DL |
| **SV-Trios Analysis Workflow** | Trio-based SV extraction, inheritance filtering & ClinVar-based annotation. | AWK · Python · BEDTools |
